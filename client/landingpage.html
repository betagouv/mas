Pourquoi api.gouv ?

Pour authentifier simplement un usager, pour le guider dans la saisie d'informations, pour ne pas lui demander des informations que l’administration possède déjà, pour personnaliser un service selon la situation de l’usager, ... autant de cas d'usage disponible via les API de l'Etat. A vous de créer d’autres usages.
